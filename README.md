# TnP Karma

**Internal Tool**: This is a dedicated, internal application for the Training & Placement cell to track volunteer hours and activity. 

## Features
- Gamified Leaderboard displaying Gold/Silver/Bronze tiers based on volunteer activity.
- Peer-verification upvote system so volunteers can mutually verify attendance.
- Secure headless database via Google Sheets API.
- "Old Reddit" styled dense, mobile-first design system with Light and Dark responsive modes.

## Deployment
Built on Next.js 15 App Router. Configured to deploy on an Ubuntu Linux droplet or Vercel.

_For internal use only by TnP Cell members._
